# WHU-Beamer-Theme
WHU 风格的 Beamer 主题，可用于答辩、汇报等

## Instructions

直接在 Overleaf 导入本项目即可，编译方式选择 $Xe\LaTeX{}$

## Features

在 THU beamer 模板上作了一点修改:

- 使用 [武汉大学](https://www.whu.edu.cn/info/1153/3232.htm) 官网发布的珞珈绿 (R:17 G:87 B:64；C:89 M:19 Y:72 K:60) 标准色
- 更新 2022 版新校徽
