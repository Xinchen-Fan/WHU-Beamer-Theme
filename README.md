# WHU-Beamer-Theme
WHU 风格的 Beamer 主题，直接在 Oerleaf 导入本项目即可

## Preview

## Features
- 在 THU beamer 模板上作了一点修改，主要更换了配色——使用 [武汉大学官网](https://www.whu.edu.cn/info/1153/3232.htm) 发布的珞珈绿 (R:17 G:87 B:64；C:89 M:19 Y:72 K:60) 标准色
- 更新了 2022 版校徽
